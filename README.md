# APUSH Website

A focused AP U.S. History study platform built for timeline review, guided notes, recall practice, and essay preparation.

## Features

- Timeline-based review
- Structured chapter notes
- Flashcards for recall
- MCQ practice
- Essay-prep tools
- Responsive UI

## Stack

- HTML
- CSS
- JavaScript

## Pages

- `index.html` — landing page
- `timeline.html` — chronology and event review
- `notes.html` — chapter notes
- `practice.html` — flashcards, quizzes, and writing practice

## Run Locally

1. Clone the repository
2. Open the project folder
3. Launch `index.html`

For smoother development, run it with a local server like Live Server.

## Purpose

This project was built to make APUSH review faster, cleaner, and more interactive than static notes alone.

## Status

In active development. More chapters, question banks, and writing tools can be expanded over time.
