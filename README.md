# APUSH Website

A structured AP U.S. History study platform designed to help students review chronology, analyze themes, and practice AP-style historical reasoning through interactive study tools.

## Overview

The APUSH Website is a front-end study resource built to make AP U.S. History review more organized and usable. It combines content navigation, timeline-based review, guided notes, flashcards, practice questions, and writing support in a single interface.

The project is designed around chapter- and source-backed study flows so students can move from factual recall to theme recognition and then to argument-based writing practice.

## Features

- Interactive home page with chapter selection and focus panels
- Timeline view for reviewing major events and historical turning points
- Notes page for structured chapter review and content navigation
- Flashcards for vocabulary and concept recall
- Multiple-choice practice for exam-style review
- Essay practice tools for thesis building and historical argument development
- Source-backed chapter data loading through modular JavaScript files
- Responsive layout for desktop and mobile use

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` – landing page and chapter overview
- `timeline.html` – interactive APUSH timeline experience
- `notes.html` – guided chapter notes and review
- `practice.html` – flashcards, quizzes, and essay practice
- `chapter-manifest.js` – chapter configuration registry
- `chapter-loader.js` – chapter loading logic
- `chapter*-data.js` – source-backed chapter content files

## Getting Started

1. Clone the repository.
2. Open the project folder locally.
3. Launch `index.html` in your browser.

For the most reliable development workflow, use a local server extension such as Live Server in VS Code.

## Purpose

This project was built to create a cleaner and more effective APUSH review experience than static notes alone. The goal is to help students study content with stronger chronological awareness, clearer thematic connections, and more deliberate exam preparation.
